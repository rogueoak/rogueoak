## Step 5 - Scroll-reveal primitive

Files: `src/components/reveal.tsx`, styles in `globals.css`.

- `"use client"` `Reveal` wraps children in a div that starts at `opacity-0 translate-y-*` and gets
  an `in-view` class via `IntersectionObserver` (unobserve after first hit).
- CSS: `.reveal { transition } .reveal.in-view { opacity:1; translate:0 }`; under
  `@media (prefers-reduced-motion: reduce)` the element is always visible with no transition.
- No-JS fallback: base style is visible; the hidden state is only applied once mounted (client), so
  server HTML shows content.

Verify: scrolling in dev reveals sections once; reduced-motion set -> no animation; JS off -> visible.
